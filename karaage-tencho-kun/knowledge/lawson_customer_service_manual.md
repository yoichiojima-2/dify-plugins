## 1. 基本理念

### ローソンの接客三原則

1. **「マチのほっとステーション」** としての自覚を持つ
2. **スピーディー** かつ **正確** なサービス提供
3. **笑顔** と **アイコンタクト** を忘れない

### 心構え

- お客様一人ひとりが「今日初めてのお客様」という気持ちで接する
- 常連のお客様にも馴れ馴れしくならない
- 忙しい時こそ丁寧な対応を心がける

## 2. 身だしなみ基準

### 制服

| 項目 | 基準 |
|------|------|
| ユニフォーム | シワ・汚れがないこと。ボタンは全て留める |
| エプロン | 店舗指定のものを正しく着用 |
| 名札 | 左胸の見やすい位置に着用 |
| 靴 | 黒または紺の動きやすい靴（サンダル不可） |

### 身だしなみ

| 項目 | 基準 |
|------|------|
| 髪 | 清潔感のある髪型。長い場合は束ねる。派手な髪色は不可 |
| 爪 | 短く切り揃える。ネイルアート・マニキュア不可 |
| アクセサリー | 結婚指輪のみ可。ピアス・ネックレス不可 |
| 香水 | 使用不可（食品を扱うため） |
| 化粧 | ナチュラルメイク推奨 |

## 3. 基本接客用語

### 必須フレーズ（接客7大用語）

| 場面 | フレーズ |
|------|----------|
| お出迎え | 「いらっしゃいませ」 |
| 感謝 | 「ありがとうございます」 |
| 承知 | 「かしこまりました」 |
| 待たせる時 | 「少々お待ちくださいませ」 |
| 待たせた後 | 「お待たせいたしました」 |
| 謝罪 | 「申し訳ございません」 |
| お見送り | 「ありがとうございました。またお越しくださいませ」 |

### 声の出し方

- **明るく** はっきりと発声する
- レジでは **お客様の目を見て** 話す
- 店内では **お客様に聞こえる声量** で挨拶する
- 深夜帯は **周囲に配慮した声量** に調整する

## 4. レジ対応

### 4.1 基本フロー

```
① お出迎え
   「いらっしゃいませ」
   ↓
② 商品スキャン
   丁寧に、かつスピーディーに
   ↓
③ ポイントカード確認
   「Pontaカード、またはdポイントカードはお持ちですか？」
   ↓
④ レジ袋確認
   「レジ袋はご利用ですか？」（有料：Sサイズ3円/Mサイズ3円/Lサイズ3円）
   ↓
⑤ 合計金額
   「合計○○円でございます」
   ↓
⑥ 支払い方法確認
   「お支払い方法はいかがなさいますか？」
   ↓
⑦ お釣り・レシート
   「○○円のお返しでございます。レシートでございます」
   ↓
⑧ お見送り
   「ありがとうございました」
```

### 4.2 支払い方法別対応

#### 現金

```
お客様：（1,000円札を出す）
店員：「1,000円お預かりいたします」
      （お釣りを渡す際）
      「○○円のお返しでございます」
```

**注意点：**
- 預かり金額は必ず復唱する
- お釣りは「お返し」、預かり金と同額の場合は「ちょうど頂戴いたします」
- 高額紙幣は必ず偽札チェッカーで確認

#### 電子マネー

```
店員：「電子マネーでございますね。こちらにタッチをお願いいたします」
      （決済音確認後）
      「ありがとうございます」
```

**対応電子マネー：**
- 交通系IC（Suica、PASMO、ICOCA等）
- 楽天Edy
- iD
- QUICPay
- WAON
- nanaco

#### QRコード決済

```
店員：「○○ペイでございますね。こちらの画面にバーコードをお願いいたします」
      または
      「こちらのQRコードを読み取ってください」
```

**対応QRコード決済：**
- PayPay
- LINE Pay
- d払い
- au PAY
- 楽天ペイ
- メルペイ

#### クレジットカード

```
店員：「クレジットカードでございますね」
      「こちらに挿入（またはタッチ）をお願いいたします」
      （暗証番号入力の場合）
      「暗証番号の入力をお願いいたします」
```

### 4.3 ポイントカード対応

#### Pontaカード

```
店員：「Pontaカードはお持ちですか？」
お客様：「はい」（カードを提示）
店員：「ありがとうございます」（スキャン）
      「○○ポイントお付けいたします」
```

#### dポイントカード

```
店員：「dポイントカードはお持ちですか？」
```

**注意：** Pontaカードとdポイントカードはどちらかのみポイント付与可能

### 4.4 レジ袋対応

```
店員：「レジ袋はご利用ですか？」

【有料を希望の場合】
店員：「サイズはいかがいたしますか？」
      「Sサイズ、Mサイズ、Lサイズ、全て3円でございます」

【不要の場合】
店員：「かしこまりました」
      （商品をカウンターに置く/マイバッグに入れるか確認）
```

## 5. 商品別対応

### 5.1 年齢確認が必要な商品

#### 対象商品
- 酒類（ビール、日本酒、ワイン、チューハイ等）
- たばこ
- 成人向け雑誌

#### 対応手順

```
店員：「恐れ入りますが、年齢確認をさせていただきます」
      「画面のボタンをタッチしていただけますか？」

【明らかに未成年に見える場合】
店員：「恐れ入りますが、身分証明書をご提示いただけますか？」

【未成年と確認された場合】
店員：「申し訳ございません。20歳未満の方への販売はできかねます」
```

**重要：**
- 年齢確認は法律で義務付けられています
- 「見た目が若く見えない」は言い訳になりません
- 迷った場合は必ず確認する

### 5.2 温め・調理が必要な商品

#### お弁当・おにぎり等

```
店員：「温めはいかがいたしますか？」

【温める場合】
店員：「少々お待ちくださいませ」
      （電子レンジで温め）
      「お待たせいたしました。熱くなっておりますのでお気をつけください」
```

**温め時間目安：**
| 商品 | 時間（500W） |
|------|-------------|
| おにぎり | 30秒 |
| 弁当（小） | 1分30秒 |
| 弁当（大） | 2分 |
| パスタ | 1分30秒 |

#### ホットスナック（からあげクン、Lチキ等）

```
店員：「からあげクンでございますね。少々お待ちくださいませ」
      （専用ケースから取り出し、袋に入れる）
      「お待たせいたしました」
```

**注意点：**
- 必ずトングを使用する（素手厳禁）
- 商品の鮮度時間を確認する（廃棄時間を過ぎたものは販売不可）

### 5.3 おでん

```
店員：「おでんでございますね。どちらになさいますか？」
      （お客様が選んだ後）
      「○○と○○でございますね」
      （専用容器に入れる）
      「からしはお付けしますか？」
```

**注意点：**
- 必ず専用のおでんトングを使用
- 汁は適量を入れる（入れすぎ注意）
- つゆだくをご希望の場合は対応する

### 5.4 中華まん

```
店員：「肉まんでございますね」
      （専用トングで取り出し、紙袋に入れる）
      「熱くなっておりますのでお気をつけください」
```

## 7. 各種サービス対応

### 7.1 公共料金・各種払込票

```
店員：「お支払いでございますね」
      （払込票のバーコードをスキャン）

      （支払い後）
      「こちらが領収書でございます。お控えは必ず保管ください」
```

**注意点：**
- 取扱期限を必ず確認
- 30万円以上は取扱不可

店員：「メルカリの発送でございますね」

## 8. クレーム対応

### 8.1 基本姿勢

1. **まず謝罪** — 内容に関わらず、まずお詫びする
2. **傾聴** — お客様の話を最後まで聞く
3. **共感** — お客様の気持ちに寄り添う
4. **対応** — 可能な解決策を提示する
5. **感謝** — ご指摘への感謝を伝える

### 8.2 よくあるケースと対応

#### レジ待ち時間が長い

```
お客様：「何でこんなに待たせるの！」

店員：「大変申し訳ございません。お待たせして申し訳ございませんでした」
      「只今、対応させていただきます」
      （他のスタッフへ）「○番レジ応援お願いします！」
```

#### 商品の品切れ

```
お客様：「○○が欲しいんだけど、ないの？」

店員：「申し訳ございません。只今、品切れとなっております」
      「次回入荷は○○の予定でございます」
      または
      「お取り寄せも可能ですが、いかがいたしますか？」
```

#### 商品の不具合・異物混入

```
お客様：「この弁当、髪の毛が入ってるんだけど！」

店員：「大変申し訳ございません。すぐに新しい商品とお取替えさせていただきます」
      「恐れ入りますが、レシートはお持ちでしょうか？」

      （商品を確認・交換後）
      「このたびは大変ご不快な思いをさせてしまい、誠に申し訳ございませんでした」
      「メーカーにも報告し、再発防止に努めてまいります」
```

**重要：**
- 異物混入の場合は、現物を保管し店長へ報告
- お客様の連絡先を伺い、後日フォローを行う場合もある

#### お釣りの間違い

```
お客様：「お釣りが足りないんだけど」

店員：「大変申し訳ございません。確認させていただきます」
      「恐れ入りますが、レシートを拝見してもよろしいでしょうか？」

      （確認後、間違いがあった場合）
      「申し訳ございません。○○円不足しておりました」
      「大変失礼いたしました」
```

### 8.3 対応できない要求への対応

```
【理不尽な値引き要求】
店員：「申し訳ございません。お値引きは致しかねます」

【禁止行為の要求（年齢確認なしでの酒・たばこ販売等）】
店員：「申し訳ございません。法律で定められておりますため、
       対応いたしかねます」

【暴言・威圧的な態度】
店員：「お客様、恐れ入りますが、そのようなお言葉は控えていただけますか」
      （改善しない場合）
      「これ以上は対応いたしかねます。お引き取りください」
      （必要に応じて店長・警察へ連絡）
```

### 8.4 店長への引き継ぎ

以下の場合は速やかに店長（または責任者）へ引き継ぐ：

- 金銭トラブル
- 重大なクレーム
- 暴言・暴力
- 法的な問題に発展しそうな場合
- 自分では判断できない場合

```
店員：「恐れ入りますが、責任者に確認いたします。少々お待ちくださいませ」
```

## 9. 防犯対応

### 9.1 万引き対応

**発見時の基本：**
- 現行犯でない限り、声をかけない
- 不審な動きは監視カメラで記録
- 店長・責任者へ報告

**現行犯の場合：**
```
店員：「恐れ入りますが、お会計がお済みでない商品があるようです」
      「こちらでお会計させていただいてよろしいですか？」
```

**注意：** 絶対に身体に触れない。否認された場合は無理に追及しない。

### 9.2 強盗対応

1. **身の安全を最優先** — 抵抗しない
2. **指示に従う** — 金銭を要求されたら渡す
3. **特徴を記憶** — 服装、体格、顔の特徴、逃走方向
4. **犯人が去ってから通報** — 110番、本部へ連絡
5. **現場保存** — 触れたものはそのままにする

### 9.3 不審者対応

```
【声かけ】
店員：「いらっしゃいませ。何かお探しですか？」
      （店内を見て回っている人への積極的な声かけ）

【明らかに不審な場合】
店員：（他のスタッフへ）「○○さん、○番通路お願いします」
      （複数人で対応）
```

### 9.4 緊急連絡先

| 状況 | 連絡先 |
|------|--------|
| 緊急通報 | 110（警察）/ 119（消防） |
| 本部緊急連絡 | 0120-XXX-XXX |
| スーパーバイザー | （店舗により異なる） |

## 10. 清掃・衛生管理

### 10.1 定時清掃

| 時間帯 | 清掃内容 |
|--------|----------|
| 開店前 | 店内全体、トイレ、駐車場 |
| 10:00 | トイレ巡回、ゴミ回収 |
| 13:00 | 店内床、レジ周り、トイレ |
| 16:00 | トイレ巡回、商品棚整理 |
| 19:00 | 店内床、ゴミ回収、トイレ |
| 22:00 | レジ周り、カウンター、トイレ |
| 深夜 | 床全面清掃（ワックス等） |

### 10.2 レジ周り

- 常にきれいな状態を保つ
- 小銭トレイは汚れたら拭く
- スキャナーは1日1回清掃
- カウンターは汚れたらすぐに拭く

### 10.3 フードケース

| 設備 | 清掃頻度 |
|------|----------|
| ホットスナックケース | 1日2回 |
| おでん什器 | 1日1回（夜間） |
| 中華まんスチーマー | 1日1回 |
| MACHI caféマシン | 1日1回 + 随時 |

### 10.4 手洗い

**以下のタイミングで必ず手洗い：**

- 出勤時
- トイレ使用後
- 休憩後
- 金銭取り扱い後に食品を触る前
- 清掃後
- ゴミ処理後

**手洗い手順：**
1. 流水で手を濡らす
2. 石鹸を付け、30秒以上洗う
3. 指の間、爪の中、手首も洗う
4. 流水で十分にすすぐ
5. ペーパータオルで拭く
6. アルコール消毒

## 付録：緊急時チェックリスト

### 地震発生時

- [ ] 身の安全を確保（カウンター下など）
- [ ] 揺れが収まったら火元確認
- [ ] お客様を安全な場所へ誘導
- [ ] 店長・本部へ連絡
- [ ] 被害状況を確認・報告

### 火災発生時

- [ ] 大声で「火事だ！」と周知
- [ ] 119番通報
- [ ] 初期消火（可能な範囲で）
- [ ] お客様・スタッフを避難誘導
- [ ] 店長・本部へ連絡

### 停電時

- [ ] 非常照明を確認
